## Algorithms

### 1. FizzBuzz
- **Path**: `fizzbuzz/{category}/{num}_{name}.js`
- **Output**: Print 1-100, one per line
  - "Fizz" if divisible by 3
  - "Buzz" if divisible by 5
  - "FizzBuzz" if divisible by both
  - Number itself otherwise

### 2. Sort
- **Path**: `sort/{category}/{num}_{name}.js`
- **Output**: Single line: `[3,9,10,27,38,43,82]`
- **Input array**: `[38, 27, 43, 3, 9, 82, 10]`
- **Algorithms used**: Bubble sort, insertion sort, selection sort, merge sort, quicksort (personality-appropriate)

### 3. Fibonacci

- **Output**: First 20 Fibonacci numbers, one per line
