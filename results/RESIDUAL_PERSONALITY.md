## Key Insight

Personality signal migrates to progressively smaller expression channels as constraints tighten:

1. **No constraints**: Full creative freedom — line count, algorithm choice, structure, naming ALL vary
2. **Readability (A)**: Line count converges, but naming remains fully expressive (themed function/variable names)
3. **Structure (B)**: Structure + line count converge, naming still leaks through loop variables
4. **Algorithm (C)**: Algorithm converges, one themed variable survives per implementation
5. **All constraints (D)**: Everything converges — personality only survives in one function name suffix

Personality is like water: it fills whatever space constraints leave open.
