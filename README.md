# AI Hello World with Different Personalities

An experiment: what happens when you ask an AI to write "Hello World" in JavaScript as 200 different fictional, historical, and celebrity personalities?

## The Experiment

200 personalities — from anime characters to scientists, villains to children's TV icons — each "wrote" a Hello World JS program. Every file is valid Node.js. The analysis script compares them statistically.

## Run

```bash
# Run any personality's Hello World
node 079_groot.js

# Run the full statistical analysis
node analyze.js
```

## Key Findings

- **All 200 files are valid Node.js** — 200/200 produce output
- Code length ranges from **7 lines** (Saitama) to **57 lines** (Rocket Raccoon)
- Character count ranges from **158** (Sasuke) to **1,521** (Rocket Raccoon)
- Complexity score ranges from **0** (Saitama, Hemingway, Iron Man) to **7** (Ethan Hunt)
